■ 简介

MiniTwitter Mod 是使用 WPF 技术的 Twitter 客户端。
你可以使用它来查看时间线和更新状态。


■ 使用环境

需要 .NET Framework 4 Client Profile ，可运行在 Windows XP/Vista/7 上


■ 安装

在适当的文件夹里安装。


■ 简单的使用方法

1. 初始化
第一次启动请进行OAuth认证，注意先设置网络环境。
登录成功就会自动更新各个时间线。

2. 添加时间线
在时间线选项卡上单击右键可以添加时间线。
可以设置绑定列表、搜索或过滤器。

3. 更改排序和外观
在时间线上右键可以改变排序方式和显示方式。

4. 关闭
如果选择常驻托盘，那么在主窗口点击关闭时会最小化到托盘。
要退出，请在托盘图标上右键然后选择退出。
